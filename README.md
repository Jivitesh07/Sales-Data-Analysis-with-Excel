# Sales-Data-Analysis-with-Excel

This project focuses on analyzing Amazon sales data using Microsoft Excel to derive meaningful business insights. The dataset is cleaned and structured to perform Pivot Table analysis, interactive dashboards using slicers, and visualizations with pivot charts. Key metrics such as total sales, profit, and profit margin are analyzed across countries, product categories, and customer segments to understand performance trends and support data-driven decision-making.

Tools Used: Microsoft Excel

Key Skills: Data Cleaning, Pivot Tables, Pivot Charts, Slicers, Business Insights, Reporting
